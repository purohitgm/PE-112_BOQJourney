
BOQ Analytics Dashboard (Streamlit)

Steps:
1. Install Python
2. Install requirements:
   pip install -r requirements.txt
3. Run:
   streamlit run app.py

Upload BOQ files for Proposal / 30 / 60 / 90 stages.

Charts include:
- MOC wise inch-dia
- Pipe / fittings / flanges / valves distribution
- Metallic vs non-metallic
- Pipe size histogram
- Top 20 lines by inch-dia
- Stage comparison
