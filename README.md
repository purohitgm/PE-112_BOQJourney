
# BOQ SharePoint Dashboard

This is a Streamlit dashboard for comparing BOQ stages (Proposal, 30%, 60%, 90%).

## Run locally

1. Install Python
2. Install requirements:

pip install -r requirements.txt

3. Run the app:

streamlit run app.py

## Deploy

Upload this repository to GitHub and deploy on a platform like Render or Azure.

## Embed in SharePoint

Add an Embed web part and paste:

<iframe src="YOUR_APP_URL" width="100%" height="900"></iframe>
